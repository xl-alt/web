3. 绘画模型使用说明

① 对话式生成 (v1/chat/completions)

通过标准对话接口即可生成图片，支持图生图（叠图）。传图格式与 OpenAI 官方文档完全一致。

注意: 图片宽高比仅支持像素比。例如1024:1024，是指宽1024，高1024像素。

在提示词末尾（提示词使用英文效果更好哦！），使用英文中括号和冒号指定宽高比，例如：
帮我画一只宇航猫在月球漫步[1024:1024]

② 专业图片生成 (v1/images/generations)

使用此接口进行文生图。负载示例如下：



{
  "model": "flux.1-kontext-dev",           // 可选，指定模型的 ID，默认使用 "MODEL_ID_IMG2IMG"
  "prompt": "a cat", // 描述图片编辑的文本，例如：替换背景
  "size": "1024x1024",             // 可选，生成图片的尺寸，默认为 1024x1024
  "response_format": "url"       // 返回格式，支持 "url"（返回图片 URL）或 "b64_json"（返回 base64 字符串）
}

③ 专业图片编辑 (v1/images/edits)

支持对单张图片进行编辑。图片可以是 URL 或 Base64 字符串。负载示例如下：


{
  "model": "flux.1-kontext-dev",           // 可选，指定模型的 ID，默认使用 "MODEL_ID_IMG2IMG"
  "image": "https://xl-files.oss-cn-chengdu.aliyuncs.com/images/img-1756543093582-76007ced-e710-4bc3-92ea-f79ff9304750.jpg",  // 要编辑的图片，可以是 base64 字符串或 URL
  "prompt": "move to water", // 描述图片编辑的文本，例如：替换背景
  "size": "1024x1024"             // 可选，生成图片的尺寸，默认为 1024x102
  "response_format": "b64_json"   // 返回格式，支持 "url"（返回图片 URL）或 "b64_json"（返回 base64 字符串）
}

4. 豆包绘画模型使用文档说明(新)

使用豆包绘画请务必参考参数说明，否则会出现空返，错误返回等情况！

如果您完全按照newapi格式对接的话请使用 doubao-seedream-4-0 模型名，如果您需要在/v1/images/generations这个路由里面叠图请使用 doubao-seedream-4-0-250828

豆包绘画模型名称为 doubao-seedream-4-0-250828 或者 doubao-seedream-3-0-t2i-250415。支持调用格式 /v1/chat/completions 的chat格式与 /v1/images/generations 和 /v1/images/edits 的dalle格式兼容。

支持的图像尺寸说明，支持输入比例，像素乘积与1k,2k的形式。其他非法输入会默认为1024x1024大小

图像生成接口支持以下三种 size 输入形式：

  • 档位形式：支持 1k、2k、4k，输入后原样返回。
  • 像素形式：例如 1280x720，必须满足以下条件：
    • 宽度范围：1280 ~ 4096
    • 高度范围：720 ~ 4096
    • 宽高比范围：[1/16, 16]，即 0.0625 ~ 16
    若输入不合法，则默认返回 1024x1024。
  • 比例形式：例如 16:9，会自动映射到推荐像素。

请求参数说明（chat格式请求参数）:

  • chat格式的请求参数统一符合 openAI 对话格式，在 content 中或者 text 中输入绘画的内容，例如：画一只猫。 在内容中通过 - 进行绘画参数配置，具体支持参数如下, 注意，各个参数之间需要有空格,不然会导致解析错误。
  • type: 绘画类型选择，目前支持两个类型，分别为 normal 和 group，默认为normal。 使用示例： -type=normal 直接放在内容中即可。normal 最大支持单次出图4张，group 最大支持单次出图10张，不指定均默认为1张图片。出图数量太多容易导致绘图失败或者中断。
  • n: 出图数量,默认为1。 使用示例： -n=2 直接放在内容中即可。限制如上。
  • watermark: 水印控制参数，默认为false。 使用示例： -watermark=false 直接放在内容中即可。如果需要生成图片右下角提示AI生成则将其设置为true即可。
  • size: 图片大小控制，默认为1k。 使用示例： -size=2k 或者按照像素 -size=1024x1024 直接放在内容中即可。点击查看支持的图片大小参数
  • 如果有底图请将其按照openAI对话传图格式即可。

chat 格式绘画完整示例:


{

请求参数说明（dalle格式请求参数）:

  • dalle 格式的请求参数统一符合 openAI dalle格式进行绘画参数配置，具体支持参数如下
  • prompt: 绘画提示词（必填）。
  • model: 绘画选择模型（必填），例如 doubao-seedream-4-0-250828 。
  • type: 绘画类型选择，目前支持两个类型，分别为 normal 和 group，默认为normal。 normal 最大支持单次出图4张，group 最大支持单次出图10张，不指定均默认为1张图片。出图数量太多容易导致绘图失败或者中断。
  • n: 出图数量,默认为1。。
  • watermark: 水印控制参数，默认为false。 如果需要生成图片右下角提示AI生成则将其设置为true即可。
  • size: 图片大小控制，默认为1k。点击查看支持的图片大小参数
  • 如果有底图请将按照dalle格式传图。
